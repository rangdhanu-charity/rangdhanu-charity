# রংধনু চ্যারিটি ফাউন্ডেশন — ডিজাইন ও প্রিন্ট রিসোর্স (LaTeX)

এই ফোল্ডারে **রংধনু চ্যারিটি ফাউন্ডেশন (Rangdhanu Charity Foundation)** এর জন্য প্রয়োজনীয় বিভিন্ন ডিজাইন ও প্রিন্টিং রিসোর্স LaTeX কোড আকারে দেওয়া হয়েছে। সবগুলো রিসোর্স সম্পূর্ণ **বাংলা ভাষায়** এবং প্রিমিয়াম ডিজাইনে তৈরি।

---

## 📂 ফাইল পরিচিতি (File Directory)

১. **`poster.tex`** - A4 সাইজের দৃষ্টিনন্দন পোস্টার। প্রচার ও প্রচারণার জন্য আদর্শ।
২. **`small_sticker.tex`** - রাউন্ডেড কর্নারযুক্ত ছোট চারকোনা স্টিকার (3" x 2")। বক্স, উপহার বা খামের উপর লাগানোর জন্য।
৩. **`rounded_sticker.tex`** - ৩ ইঞ্চি ব্যাসের বড় গোলাকার স্টিকার। ব্র্যান্ডিংয়ের জন্য।
৪. **`rounded_small_sticker.tex`** - ১.৫ ইঞ্চি ব্যাসের ছোট গোলাকার স্টিকার। খাম সিল (Envelope Seal) বা ছোট গিফট প্যাকেজিংয়ের জন্য।
৫. **`envelope.tex`** - রংধনু থিমের কাস্টমাইজড লেটার খাম (DL সাইজ: 110mm x 220mm)। প্রেরক ও প্রাপকের ঠিকানা সহ।
৬. **`Logo_bright.png`** - সংস্থার মূল লোগো (LaTeX ফাইলে ইমেজ হিসেবে ব্যবহার করা হয়েছে)।

## 🛠️ প্রয়োজনীয় সফটওয়্যার ও সেটআপ (Setup Guide)

যেহেতু ফাইলগুলো জটিল বাংলা যুক্তবর্ণ ব্যবহার করে তৈরি, তাই এগুলোকে কম্পাইল করার জন্য **XeLaTeX** (বা LuaLaTeX) ইঞ্জিন ব্যবহার করতে হবে। সাধারণ PDFLaTeX ব্যবহার করলে বাংলা ফন্ট সঠিকভাবে আসবে না।

### ১. LaTeX কম্পাইলার ইনস্টলেশন
- **Windows (উইন্ডোজ):** [MiKTeX](https://miktex.org/) অথবা [TeX Live](https://www.tug.org/texlive/) ইনস্টল করুন। সঙ্গে **TeXworks** বা **VS Code (LaTeX Workshop extension)** এডিটর ব্যবহার করতে পারেন।
- **macOS (ম্যাক):** [MacTeX](https://www.tug.org/mactex/) ইনস্টল করুন।
- **Linux (লিনাক্স):** `texlive-full` এবং `texlive-lang-indic` প্যাকেজগুলো ইনস্টল করুন।

### ২. বাংলা ফন্ট ইনস্টলেশন
এই ডিজাইনগুলোতে গুগলের ফ্রি ওপেন-সোর্স **Noto Sans Bengali** ফন্টটি ব্যবহার করা হয়েছে। ফন্টটি পিসিতে ইনস্টল না থাকলে বাংলা লেখা আসবে না।
- **Noto Sans Bengali** ফন্টটি ডাউনলোড করতে এখানে যান: [Google Fonts - Noto Sans Bengali](https://fonts.google.com/specimen/Noto+Sans+Bengali)
- ফন্ট ফাইলটি (.ttf) ডাউনলোড করে আপনার কম্পিউটারে সাধারণ ফন্ট যেভাবে ইনস্টল করে (ডাবল ক্লিক করে Install) সেভাবে ইনস্টল করে নিন।
- *বিকল্প ফন্ট:* আপনি চাইলে **Kalpurush** (কালপুরুষ) বা **SolaimanLipi** (সোলাইমান লিপি) ফন্টও ব্যবহার করতে পারেন। সেক্ষেত্রে প্রতিটি `.tex` ফাইলের শুরুর দিকে ফন্টের নাম পরিবর্তন করে নিতে হবে:
  ```latex
  \setmainfont[Script=Bengali]{Kalpurush}
  ```

## 🚀 পিডিএফ তৈরির নিয়ম (How to Compile)

টার্মিনাল বা কমান্ড প্রম্পটে (Command Prompt) এই ফোল্ডারে প্রবেশ করে নিচের কমান্ডগুলো রান করলেই প্রিমিয়াম কোয়ালিটির ভেক্টর PDF ফাইল তৈরি হয়ে যাবে:

### ১. এ৪ পোস্টার (A4 Poster) তৈরি করতে:
```bash
xelatex poster.tex
```

### ২. ছোট চারকোনা স্টিকার (Small Sticker) তৈরি করতে:
```bash
xelatex small_sticker.tex
```

### ৩. বড় গোলাকার স্টিকার (Large Rounded Sticker) তৈরি করতে:
```bash
xelatex rounded_sticker.tex
```

### ৪. ছোট গোলাকার স্টিকার (Small Rounded Sticker) তৈরি করতে:
```bash
xelatex rounded_small_sticker.tex
```

### ৫. কাস্টম খাম (Letter Envelope) তৈরি করতে:
```bash
xelatex envelope.tex
```

> 💡 **পরামর্শ:** প্রথমবার রান করার সময় কম্পাইলার প্রয়োজনীয় প্যাকেজগুলো নিজে থেকেই ডাউনলোড করে নিতে পারে (যেমন `tcolorbox`, `tikz` ইত্যাদি)। তাই ইন্টারনেট কানেকশন চালু রাখার পরামর্শ দেওয়া হচ্ছে।
